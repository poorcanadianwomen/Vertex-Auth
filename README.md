# Vertex Auth 🔐

**Secure Two-Factor Authentication for VertexSender**

Vertex Auth is a community-driven security tool designed for users of [VertexSender](https://www.vertexsender.com/). This project aims to provide a lightweight, open-source Android application that generates Time-based One-Time Passwords (TOTP) to add an extra layer of protection to your VertexSender account.

## 🚀 What is Vertex Auth?

Vertex Auth is a dedicated Two-Factor Authentication (2FA) app built specifically for the [VertexSender](https://www.vertexsender.com/) community. It allows you to secure your account using time-based codes generated directly on your device.

### Why use Vertex Auth?

*   **Enhanced Security:** Protect your [VertexSender](https://www.vertexsender.com/) account from unauthorized access.
*   **Open Source:** Transparent codebase that anyone can audit for security and trust.
*   **Lightweight:** No bloatware, no unnecessary permissions, and minimal battery usage.
*   **Privacy Focused:** Codes are generated locally on your device. No internet connection is required to generate tokens.
*   **Free:** Completely free to use, with no subscriptions or hidden fees.

## ⚙️ How It Works

1.  **Enable 2FA on VertexSender**
    *   Log in to your [Vertex Account](https://www.vertexsender.com/login).
    *   Navigate to **Settings** > **Security**.
    *   Select **2FA Enabled**.

2.  **Connect Vertex Auth**
    *   Once the app is released, open Vertex Auth on your phone.
    *   Scan the QR code displayed.

3.  **Start Securing Your Account**
    *   Vertex Auth will generate a 6-digit code that changes every 30 seconds.
    *   Enter your code to verify your session.
    *   and boom your done :3
